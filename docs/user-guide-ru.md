# Руководство пользователя (AJAZZ AJ179 APEX Battery Monitor)

## Быстрый запуск
1. Скачайте `AjazzBatteryMonitor-win-x64.exe` или `AjazzBatteryMonitor-win-x64-portable.zip` из папки `artifacts/`.
2. Запустите `AjazzBatteryMonitor-win-x64.exe`. Права администратора не требуются.
3. В системном трее Windows появится значок с процентом заряда батареи (например, `74`).

## Значения значков в системном трее
- **Число (0–100)**: Реальный процент заряда батареи.
- **⚡ (Молния)**: Мышь подключена к источнику питания и заряжается.
- **? (Знак вопроса)**: Заряд неизвестен (устройство отключено или не отвечает).
- **Zzz**: Мышь находится в режиме энергосбережения / сна.
- **X**: Устройство не подключено.

## Контекстное меню трея (Клик правой кнопкой мыши)
- **Открыть сведения**: Открывает главное окно приложения с историей и настройками.
- **Обновить сейчас**: Принудительно выполняет мгновенный опрос мыши.
- **Запуск вместе с Windows**: Включает/выключает автозапуск приложения через реестр текущего пользователя.
- **Выход**: Завершает работу приложения.

## Совместимость с официальным приложением AJAZZ
Если официальная утилита AJAZZ открыта и удерживает доступ к устройству, приложение покажет предупреждение в трее и подсказку:
*"Официальное приложение AJAZZ заняло интерфейс устройства. Закройте его или повторите попытку позже."*
Приложение не завершается аварийно.
